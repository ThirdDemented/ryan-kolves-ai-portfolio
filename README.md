## Evidence Drift Lab

The portfolio now includes a working, browser-local adaptive evaluation prototype. It classifies claims by epistemic status, runs a bounded evaluator–critic–optimizer loop, exposes each iteration and stop condition, and updates local scoring weights from human feedback. The demonstration accepts only public, fictional, or non-sensitive text and sends no input to an external service.
